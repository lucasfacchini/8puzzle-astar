# Description #

An implementation for solving the 8 puzzle game using A* algorithm, written in Python.

This is an assignment submission for Artificial Intelligence course (Universidade de Caxias do Sul), made by [Lucas Facchini](https://github.com/lucasfacchini) and [Adryan Alessandro Signor](https://github.com/AdryanSignor).

# Running #

```
python eight_puzzle_test.py
```
Note: Python 3.6 is required.
